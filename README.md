## Struktur Proyek
Proyek ini diorganisir secara modular agar mudah dipelihara:
- `/solvers`: Berisi algoritma inti (Thomas Algorithm, Cholesky Decomposition, Iterasi Gauss-Seidel/Jacobi).
- `/penyelesaian`: Berisi skrip eksekusi untuk setiap bagian soal (11.1 - 11.28).
- `main.py`: File utama untuk menjalankan seluruh penyelesaian soal secara berurutan.
